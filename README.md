## Requirements
- npm

## How to install
  ```
  git clone git@github.com:Otooo/warriorjs-levels.git (or git clone https://github.com/Otooo/warriorjs-levels.git)
  git checkout -b myHero(Your name hero)
  npm install --local @warriorjs/cli
  ```

## How to execute the game
  ```
  npx warriorjs
  ```
